📊 Customer Segmentation & Discount Optimization

Unlocking revenue growth through data-driven customer insights and strategic discount allocation.

🚀 Project Overview

This project focuses on analyzing customer behavior to identify distinct customer segments and optimize discount strategies. By leveraging data analytics, we aim to improve marketing effectiveness, increase customer engagement, and maximize revenue.

📌 Key Objectives

Segment customers based on purchasing behavior

Analyze seasonal sales trends

Identify top-performing products

Evaluate discount usage across customer clusters

Provide actionable business insights
